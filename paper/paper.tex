\section{Introduction}

This study replicates \textbf{High-Dimensional Metrics in R} written by \textbf{Victor Chernozhukov, Christian Hansen, and Martin Spindler} and published in \textbf{The R Journal in 2016} \citep{chernozhukov2016hdm}. The paper studies high-dimensional econometric methods for prediction and causal inference when the researcher has many possible control variables.

The contribution of this paper is that it gets around the following challenge: \textbf{standard regression methods can become unreliable when there are many possible controls relative to the sample size, especially when the researcher wants valid inference after variable selection}. It does this by \textbf{using rigorous LASSO and Post-LASSO methods with orthogonalized estimating equations for valid causal inference in high-dimensional settings}.

This replication focuses on \textbf{Section 6.3, the 401(k) plan participation application} from the original paper. Specifically, the paper reports that \textbf{the estimated average treatment effect (ATE) is 10,180.09 with a standard error of 1,930.68, and the estimated average treatment effect on the treated (ATET) is 12,628.46 with a standard error of 2,944.43}.


\section{Data}

The data are \textbf{the pension dataset included in the hdm R package} and contain information on \textbf{401(k) participation, 401(k) eligibility, household wealth, income, age categories, family size, education, marital status, home ownership, IRA participation, and other household characteristics}. The sample includes \textbf{9,915 observations}.

The data were originally collected from \textbf{the 1991 Survey of Income and Program Participation (SIPP)}. The key variables used in the replication are \textbf{total wealth, 401(k) participation, income category controls, age category controls, family size, education indicators, marital status, two-earner household status, defined benefit pension status, IRA participation, and home ownership}.

The original code is written in \textbf{R} and this replication reproduces the analysis in \textbf{R}.

\section{Methods}

The primary method is \textbf{high-dimensional treatment effect estimation using rigorous LASSO/Post-LASSO}, which does \textbf{variable selection among many possible controls while still allowing valid inference for the treatment effect}. The treatment effect methods are related to post-selection inference methods developed by \citet{belloni2014treatment}.

The method predicts/identifies \textbf{the effect of 401(k) participation or eligibility on total wealth} by exploiting \textbf{observed household-level covariates and LASSO-based adjustment for confounding factors}.

The estimating equations used in the paper are shown below:

\begin{equation}
ATE = E[Y(1)-Y(0)]
\end{equation}

\begin{equation}
ATET = E[Y(1)-Y(0)\mid D=1]
\end{equation}

where:
\begin{itemize}
    \item $Y_i$ is \textbf{total wealth}
    \item $D_i$ is \textbf{the treatment indicator for 401(k) participation or eligibility}
    \item $X_i$ is \textbf{the set of household-level controls used in the pension application}
\end{itemize}

The main thing to point out is \textbf{the replicated ATE and ATET estimates match the values reported in the original paper}.

This result tells us in plain English that \textbf{401(k) participation or eligibility is associated with higher total wealth after adjusting for observed covariates using high-dimensional treatment effect methods}. The ATE describes the average effect for the full population, while the ATET describes the average effect for the treated group.

Compared to the original paper, the replicated ATE and ATET estimates are \textbf{the same} as the published estimates. Possible reasons for this include \textbf{the hdm package includes the same pension dataset and the paper provides directly reproducible R code for the 401(k) application}.

\section{Challenges in Replication}

The challenges in replicating the results were the following:

\begin{itemize}
    \item \textbf{Understanding which section of the paper was narrow enough to replicate}
    \item \textbf{Distinguishing between ATE, ATET, LATE, and LATET}
    \item \textbf{Identifying the correct dataset, since the paper contains several empirical applications}
    \item \textbf{Making sure the R code uses the same simple model specification as the paper}
\end{itemize}

One important lesson from the replication process is that \textbf{a paper can be easier to replicate when the authors provide the data, code, and package functions in one place, but the researcher still needs to carefully identify the exact result and model specification being replicated}.


\section{Takeaways}

The following would be the main takeaways of the paper:

\begin{enumerate}
    \item \textbf{LASSO and Post-LASSO can be used for valid inference in high-dimensional econometric settings}
    \item \textbf{Orthogonalized treatment effect methods help estimate causal parameters after variable selection}
    \item \textbf{In the 401(k) application, the estimated ATET is larger than the ATE, meaning the estimated effect is larger for the treated group than for the full sample}
\end{enumerate}

Overall, the paper contributes to the literature by \textbf{providing an accessible R package for high-dimensional econometric estimation, inference, instrumental variables, and treatment effects}.


\section{Extensions}

The following would be possible extensions of the paper:

\begin{itemize}
    \item \textbf{Replicate the LATE and LATET estimates using 401(k) eligibility as an instrument}
    \item \textbf{Compare the simple model to the model with all interaction effects}
    \item \textbf{Compare the hdm treatment effect estimates to simpler OLS or matching-based estimates}
\end{itemize}

Future work could also explore \textbf{how sensitive the 401(k) treatment effect estimates are to alternative control sets, bootstrap standard errors, or different measures of wealth}.
