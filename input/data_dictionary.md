# Data Dictionary

This project uses the `pension` dataset from the `hdm` R package. The replication focuses on the 401(k) participation application in Chernozhukov, Hansen, and Spindler (2016), Section 6.3.

Variables used:

- `tw`: total wealth, main outcome.
- `p401`: treatment indicator for 401(k) participation.
- `e401`: 401(k) eligibility, used as an instrument for a possible LATE/LATET extension.
- `i2`, `i3`, `i4`, `i5`, `i6`, `i7`: income category indicators.
- `a2`, `a3`, `a4`, `a5`: age category indicators.
- `fsize`: family size.
- `hs`: high school indicator.
- `smcol`: some college indicator.
- `col`: college indicator.
- `marr`: married indicator.
- `twoearn`: two-earner household indicator.
- `db`: defined benefit pension indicator.
- `pira`: IRA participation indicator.
- `hown`: home ownership indicator.
